# Prestige Garage — Visual DNA (Master Art Direction)

Source of truth for every image, video, animation, and UI color used on the
cinematic marketing site. Do not deviate without updating this file first.

## Primary palette (exact values)

| Token | Hex | Use |
|---|---|---|
| Ultra Deep Black | `#050505` | Main background |
| Matte Black | `#0D0D0D` | Secondary background / cards |
| Deep Crimson Red | `#7A0018` | Primary accent (logo red, CTAs, rim light) |
| Dark Blood Red | `#5B0012` | Secondary accent (hover, deep shadow blend) |
| Metallic Silver | `#C8C8C8` | Metal accents, chrome highlights |
| Light Gray | `#A0A0A0` | Secondary text |
| Pure White | `#FFFFFF` | Primary text, steam, smoke |

## Golden rule

No color outside this identity appears unless it has a real functional
purpose in the scene — not decoration:

- Smoke / steam → always white or light gray. **Never red-tinted.**
- Nano ceramic liquid → **silver / mercury / liquid-chrome**, never red,
  never plain water.
- PPF film → almost invisible; visible only via light reflection and edge
  highlight, never as a colored film.
- Heat (thermal insulation section only) → natural orange/red glow is
  allowed here specifically because it represents heat, not brand color.
- Water → transparent.
- Sparks (rare, sparingly) → warm gold, never as a main effect.
- The vehicle is always the same deep gloss black car across every scene —
  never swapped, never decaled.
- The logo is always the real Prestige Garage crown logo file — never
  redrawn, simplified, or approximated by a generator.

## Style references (never)

Not a restaurant, not a gaming site, not a tech startup, not a corporate
dashboard. Always: luxury automotive showroom / premium European car
studio / high-end protection laboratory.

## Camera language

Slow push, elegant orbit, smooth dolly, macro detail, soft rotation.
Never handheld, never shaky, never an aggressive action-scene feel.

## Emotional target

Porsche / Rolls-Royce / Mercedes-Maybach / Pagani digital-experience tier.
The visitor should feel the craftsmanship before they ever see a price.
